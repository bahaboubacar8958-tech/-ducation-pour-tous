# Groupe Scolaire CHEICK CHERIF SAGALE Website

## Overview
This project is a website designed for the Groupe Scolaire CHEICK CHERIF SAGALE, aimed at providing educational resources for students in the 7th, 8th, 9th, and 10th grades. The website features content in French, focusing on various aspects of language learning including vocabulary, conjugation, grammar, spelling, written expression, and image study.

## Project Structure
The project is organized as follows:

```
Cours de Francais de Mr Romain/
├── index.html              # Main entry point of the website
├── styles/
│   ├── style.css           # Main styles for the website
│   ├── layout.css          # Layout-specific styles
│   └── responsive.css      # Responsive design styles
├── scripts/
│   ├── app.js              # Main JavaScript functionality
│   ├── navigation.js       # Navigation logic
│   └── interactive.js      # Interactive features
├── pages/
│   ├── grade7.html         # Content for 7th grade
│   ├── grade8.html         # Content for 8th grade
│   ├── grade9.html         # Content for 9th grade
│   ├── grade10.html        # Content for 10th grade
│   ├── vocabulary.html     # Vocabulary resources
│   ├── conjugation.html    # Conjugation resources
│   ├── grammar.html        # Grammar resources
│   ├── spelling.html       # Spelling resources
│   ├── written-expression.html # Written expression resources
│   └── image-study.html    # Image study resources
├── assets/
│   └── logo.jpeg           # Logo and other images
├── README.md               # Project documentation
└── package.json            # NPM configuration file
```

## Features
- **Grade-Specific Content**: Tailored resources for 7th to 10th-grade students.
- **Interactive Learning**: Engaging features such as quizzes and exercises.
- **Responsive Design**: Optimized for various devices and screen sizes.
- **Comprehensive Language Resources**: Covers vocabulary, grammar, conjugation, spelling, written expression, and image study.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies using npm:
   ```
   npm install
   ```
4. Open `index.html` in your web browser to view the website.
